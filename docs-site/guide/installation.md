# Installation

## Core runtime

```bash
pnpm add a2a-mesh
```

## Common stacks

```bash
pnpm add a2a-mesh a2a-mesh-adapters
```

## Optional companion tools

```bash
pnpm add a2a-mesh-registry a2a-mesh-cli
```

## Requirements

- Node.js 22.13 or newer
- TypeScript 5.x for the best DX
- optional provider SDKs such as `openai`, `@anthropic-ai/sdk`, or `langchain`
