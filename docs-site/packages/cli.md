# `a2a-mesh-cli`

Install globally or run through `pnpm dlx`:

```bash
pnpm dlx a2a-mesh-cli validate http://localhost:3000
pnpm dlx a2a-mesh-cli monitor http://localhost:3000
pnpm dlx a2a-mesh-cli benchmark http://localhost:3000
pnpm dlx a2a-mesh-cli export-card http://localhost:3000 --output agent-card.json
```
