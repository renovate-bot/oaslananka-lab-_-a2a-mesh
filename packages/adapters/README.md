# a2a-mesh-adapters

Framework adapters for running A2A agents on top of popular provider and orchestration SDKs.

## Supported adapters

- OpenAI
- Anthropic
- LangChain
- Google ADK
- CrewAI
- LlamaIndex

## Install

```bash
pnpm add a2a-mesh-adapters
```

Install only the peer SDKs you need for your chosen adapters.
